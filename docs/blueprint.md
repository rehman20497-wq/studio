# **App Name**: USA Testimonial Network

## Core Features:

- Animated USA Map: Dynamically render a glowing USA map with animated lines connecting states, serving as the backdrop for testimonials.
- Profile Placement: Display reviewer profile pictures on the map, anchored to their respective state locations for geographic accuracy.
- Sequential Profile Display: Animate profiles one by one with a smooth zoom-in effect and a circular stroke that connects them sequentially.
- Popover Reviews: Show floating popovers with glowing borders and glassmorphism style, displaying randomized review text, reviewer name, and state.
- Review Duration: Control how long popover review is visible using timer based duration tool.
- Grouped Sequences: Fading sequence which displays 5 testimonials.
- Continuous Transition: Move the animation seamlessly into the next set of testimonials.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) for a futuristic and trustworthy feel.
- Background color: Dark navy (#1A237A) with 20% saturation to allow the neon colors to pop.
- Accent color: Vibrant magenta (#FF69B4), creating a sharp contrast against the electric blue.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Neon-style icons with simple shapes to represent states and user profiles.
- Full-screen layout with the USA map as a central element and profiles positioned according to state locations.
- Smooth easing and cinematic timing for all transitions, with neon strokes, glowing points, and animated map lines for a connected, flowing experience.
